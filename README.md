altai-crop-analysis.py - данный скрипт на Python предназначен для комплексного анализа влияния климатических факторов (температура, осадки) и вегетационного индекса (NDVI) на урожайность сельскохозяйственный культур.
Final_Agroclimate_Analysis.xlsx - результат выполнения, содержащий годовые данные по Урожайности, NDVI, CAT, Осадкам и Классу Увлажнения.
